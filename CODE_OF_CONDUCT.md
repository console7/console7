# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, as its Code of Conduct.
The full text — including the standards of expected behaviour and the enforcement
guidelines — is published canonically at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project in any way — contributing code, opening issues,
reviewing, or taking part in discussion — you agree to uphold it.

## Scope

This Code applies within all project spaces (the repository, issues, pull requests,
and discussions) and applies when an individual is officially representing the project
in public spaces.

## Reporting and enforcement

Report any concern privately to the maintainers at **security@naanya.biz**. All
reports are reviewed and investigated promptly and fairly. The maintainers are
obligated to respect the privacy and security of anyone who reports an incident, and
will apply the Contributor Covenant's Enforcement Guidelines (linked above) when
determining consequences.

Maintainers are accountable under the project's [governance](GOVERNANCE.md); a
maintainer who does not uphold or enforce this Code in good faith may face
repercussions determined by the other maintainers.
